# 🦄 KuKuMBA — модульная казино-платформа

Няшное, но серьёзное казино с **одной честной игрой** — рулеткой с **provably-fair** RNG и
**RTP ровно 99%**. При этом внутри — полный набор функций современного онлайн-казино:
кошелёк (демо + реал, фиат + крипта), депозиты/выводы, бонусы, промокоды, рефералка, VIP,
кешбэк, розыгрыши, чат, уведомления, KYC, поддержка, ответственная игра и админ-панель.

Дизайн — пастельная радуга в духе My Little Pony на тёмной футуристичной базе, glassmorphism,
без пружинистых анимаций.

> ⚠️ **О реальных деньгах.** Архитектура поддерживает реальные деньги (фиат и крипта), но по
> умолчанию подключён **sandbox-провайдер платежей** — реальные средства не двигаются. Запуск с
> реальными деньгами требует подключения настоящего платёжного провайдера, лицензии на азартные
> игры и соблюдения KYC/AML в вашей юрисдикции. Это ответственность владельца.

## Возможности

- 🎯 **Рулетка** — европейское колесо (37 ячеек), provably-fair (HMAC-SHA256), **RTP 99%**.
  Множитель каждой ставки выводится из RTP: `payout = RTP / probability` → house edge ровно 1%.
- 💼 **Мультивалютный кошелёк** — режимы **DEMO** и **REAL**; валюты: `DEMO, USD, EUR, RUB,
  BTC, ETH, USDT, TON, TRX, SOL, BNB, XMR`. **USDT мультисетевой** (ERC20/TRC20/TON/SOL/BSC).
  Все движения — через append-only **ledger** с блокировкой строк (FOR UPDATE).
- 💳 **Депозиты/выводы** — модульные адаптеры (`PaymentProvider`); по умолчанию `MockProvider`.
- 🎁 **Бонусы**, 🏷 **промокоды**, 🤝 **рефералка** (комиссия с каждой ставки),
  👑 **VIP** (XP → уровни, рейкбэк), 💸 **кешбэк** от чистых потерь.
- 🎉 **Розыгрыши** — от админа/партнёров; выбор суммы и **кол-ва победителей**, колесо из
  участников, честный (provably-fair) выбор победителей.
- 💬 Чат, 🔔 уведомления, онлайн-счётчик и **лента живых ставок** (Socket.IO).
- 🪪 KYC, 🛟 поддержка (тикеты + FAQ), 🛡 ответственная игра (лимиты, самоисключение).
- ⚙️ **Админ-панель**, подключённая напрямую к БД: пользователи, балансы, выводы, промокоды,
  бонусы, розыгрыши, валюты, настройки (включая RTP), контент-страницы, аудит.
- 🌍 Интерфейс **RU + EN** (i18next).

## Технологии

| Слой        | Стек                                                              |
|-------------|------------------------------------------------------------------|
| Backend     | NestJS (модульный) · Prisma · PostgreSQL · Socket.IO · JWT       |
| Frontend    | React · Vite · TypeScript · Tailwind CSS · TanStack Query · Zustand |
| Realtime    | Socket.IO (онлайн, живые ставки, чат, пуш уведомлений)           |
| Монорепо    | pnpm workspaces                                                  |

Каждая фича — **отдельный NestJS-модуль** (`apps/api/src/modules/*`), что и обеспечивает
модульность для быстрых правок и расширения.

## Запуск (локально)

### 1. Зависимости
- Node ≥ 20, pnpm, PostgreSQL 14+, Redis (опционально — для масштабирования realtime).

```bash
pnpm install
```

### 2. Окружение
```bash
cp .env.example .env
# отредактируйте DATABASE_URL, секреты JWT и т.д.
```

### 3. База данных
```bash
pnpm prisma:generate     # сгенерировать Prisma Client
pnpm db:push             # применить схему к БД
pnpm db:seed             # наполнить (валюты, игра, VIP, админ, контент, промокоды, розыгрыш)
```

### 4. Старт
```bash
pnpm dev                 # запускает api (:3000) и web (:5173) параллельно
# или по отдельности:
pnpm dev:api
pnpm dev:web
```

Откройте **http://localhost:5173**. API — на **http://localhost:3000/api**.

### Учётки после сидов
- **Админ:** `admin@kukumba.local` / `admin12345`
- **Игроки:** `PinkiePie` / `player123`, `TwilightSpark` / `player123`
- **Промокоды:** `KUKUMBA`, `WELCOME50`, `VIPBOOST`

> Новый пользователь при регистрации получает **демо-баланс** и уникальный **Account ID**.

### Сети с ограниченным доступом (Prisma engines)
Если загрузчик движков Prisma не может скачать бинарники (корпоративный прокси и т.п.),
скачайте их напрямую и укажите через переменные окружения:
```bash
export PRISMA_QUERY_ENGINE_LIBRARY=/path/libquery_engine-<platform>.so.node
export PRISMA_SCHEMA_ENGINE_BINARY=/path/schema-engine-<platform>
# при необходимости: export NODE_EXTRA_CA_CERTS=/path/ca-bundle.crt
```

## Как устроена честность и RTP

1. Сервер генерирует `serverSeed`, публикует `SHA256(serverSeed)` **до** ставки.
2. У игрока есть `clientSeed` (можно менять) и `nonce` (растёт с каждым спином).
3. Исход: `floor( f(HMAC_SHA256(serverSeed, "clientSeed:nonce")) × 37 )`.
4. При смене сида старый `serverSeed` раскрывается — можно перепроверить всю историю
   (эндпоинт `POST /api/provably-fair/verify`).
5. **RTP 99%** задаётся выплатами: `multiplier(bet) = RTP / probability(bet)` — матожидание
   любой ставки равно `ставка × 0.99`. RTP меняется в настройках (`game.rtp`) без передеплоя.

## Платежи: sandbox → реальный провайдер
Реализуйте интерфейс `PaymentProvider` (`apps/api/src/modules/payments/providers/`) для
нужного крипто-шлюза/фиат-PSP и подключите его в `payments.module.ts` вместо `MockProvider`.
Остальная система не меняется.
